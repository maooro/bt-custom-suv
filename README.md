# BT Machine

Custom Bluetooth machine designed entirely from scratch, including both the machine itself and its dedicated controller.

## Overview
This project is a fully custom-designed Bluetooth-based machine, developed as a personal engineering project.  
It includes both the hardware and firmware components for the machine and its controller.  
The system was designed with a focus on reliability, modularity, and clear architecture.

## Features
- Fully custom-designed Bluetooth machine
- Complete custom controller designed from scratch
- Modular hardware design for both machine and controller
- Low power consumption and efficient communication
- Easy firmware updates and maintainable code
